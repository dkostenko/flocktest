# flocktest

Test task solution.

## Installation

```
$ lein compile
$ lein uberjar
```

## Usage

```
Usage: java -jar ./target/flocktest-1.0.0-standalone.jar [OPTIONS]
or: lein run [OPTIONS]

Options:
  --port                        Port number for public API (default 3000)
  --tpsize                      Threadpool size (default 1)
```
